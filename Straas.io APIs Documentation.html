Straas.io APIs Documentation

Quick Start

To get started, you should have a Straas.io account and create an application for your service. If you haven’t had an account, please contact us to create one and also create an application for you. After the application is created, you will receive a uid and a secret.

Making an API call

We use token based authentication. The BEARER token should be provided in an Authorization header for every account API request.

The AccountService section shows you how to exchange an account service token with the uid and the secret.

Create Member

Every straas.io member has his own resource access right and his purchase records. You have to create a relation between your members and straas.io members via the Create NEW Member API before you use any API related to a new member. It is better to construct the relation immediately when your new member signs up completely.

After you create the relation, you can call the APIs with your member id. For example, call GET /api/v1/account/members/:member_id with member_id=”John12345”, your members’s straas.io information such as orders and memberships will be returned.

Embed Player

Video

  • embed url

    https://app.straas.net/[account_id]/videos/[video_id]

  • params

Name Value Meaning Mandatory Note
autoplay [1, 0] If set to 1, video will autoplay no default 0 (false)
loop [1, 0] If set to 1, video will loop no default 0 (false)

  • embed html example

    <iframe frameborder="0" allowfullscreen="allowfullscreen" width="640" height="360" src="https://app.straas.net/straas-dev/videos/zKhrsFP6?autoplay=1&loop=1"></iframe>

Playlist

  • embed url

    https://app.straas.net/[account_id]/playlists/[playlist_id]

  • params

Name Value Meaning Mandatory Note
autoplay [1, 0] If set to 1, playlist will autoplay no default 0 (false)
loop [1, 0] If set to 1, playlist will loop no default 0 (false)
menu [1, 0] If set to 1, it will show playlist menu no default 1 (true)

  • embed html example

    <iframe frameborder="0" allowfullscreen="allowfullscreen" width="940" height="360" src="https://app.straas.net/straas-dev/playlists/v5WchcLs?autoplay=1&loop=1&menu=1"></iframe>
