Quick Start

To get started, you should have a StraaS account and create an application for your service. If you haven’t had an account, please contact us to create one and also create an application for you. After the application is created, you will receive a client id and a client secret.

Please refer to the Technical Document for more development details.

Get Monetization Rules

Get the monetization rules of the monetizable item such as a video or a playlist.

Endpoint

GET /api/v1/app/monetization_rules

Parameters

Name Description
type Get all monetization rules related to the monetization type. You can use type="membership_type" to get the subscription choise. {video, playlist, membership_type}
video_ids Get all monetization rules related to these videos.
playlist_ids Get all monetization rules related to these playlists.
page Page of results to fetch
per_page Number of results to return per page
sort Data sorted with {+,-} by which column {monetizable_id}. Apply - to sort in descending order. For example, sort=-column
